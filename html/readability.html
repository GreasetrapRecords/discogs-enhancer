Tracklist Readability Settings

These settings determine when dividers are inserted into a release's track list.

You must refresh your page on Discogs for changes to take effect.

Vinyl, Cassette, Multi-CD, Box Set Releases

Enabled:

This will insert readability dividers when the Vinyl or Cassette release exceeds the number of tracks listed here. If the release does not exceed this threshold, no dividers will be inserted.

Single CDs, Digital Releases

Enabled:

This will insert readability dividers when any non-prefixed release exceeds the number of tracks listed here. If the release does not exceed this threshold, no dividers will be inserted.

This will insert readability dividers every nth track. For instance, if you select 5, a divider will be inserted after track 5, 10, 15, 20, etc...
