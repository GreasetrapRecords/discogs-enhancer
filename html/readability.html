Tracklist Readability Settings

These settings determine when dividers are inserted into a release's track list in order to improve the readability. You can show and hide readability dividers by clicking "Show/Hide Dividers" at the top of the track list. Your show/hide preference will be saved each time you change it. You must refresh your page on Discogs for changes to take effect.

How it works

There are some requirements that must be met before any dividers are inserted into the track list.

  • It wont insert dividers when there are Index Tracks or Track Headings already present on the release. These releases are usually readable on their own.
  • There must be sufficient tracks present before it does anything. The minimum number of tracks is determined by the Threshold setting. Once the Threshold has been exceeded, then the dividers will be inserted. For example, if you have a Threshold value of 8, any release with 9 or more tracks will have dividers inserted.
  • If there are vinyl tracks listed like: ABCDE (ie: there are no numbers after the side delineation; essentially one track per side), it won't insert spacers between those. It will insert tracks between sides when there are multiple tracks per side (eg: A1, A2, B, C1, C2, etc …). Here is a good example of this rule.
  • Multi-CD releases must have their tracks prefixed with CD1-, 1-, or 1., for it to detect different discs. Assuming tracks are correctly listed, dividers will be inserted between each disc. Here is an example of this rule.
  • Note: There is no single standard when it comes to listing releases with multiple formats and CDs. Because of this, there is no guarantee that the dividers will be correctly inserted for every release. I've attempted to cover the most common prefixes I've found, but there are others that have not been accounted for. If you find a release that does not work well with this feature, please let me know and I'll try to get it fixed.

Single CDs, Digital Releases, etc …

Enabled:

This will insert readability dividers when the release has only sequential numbers for track positions. This is most commonly seen in single CD releases or digital releases (eg: 1, 2, 3, 4, 5, 6, 7, etc …)
